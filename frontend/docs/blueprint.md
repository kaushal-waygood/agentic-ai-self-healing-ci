# **App Name**: CareerPilot

## Core Features:

- Profile Creation: User onboarding flow with profile setup, capturing job preferences and experience.
- CV Generation: AI-powered CV generation from uploaded documents or structured form input, available in different tiers.
- Content Generation: Tailored cover letter and email drafts generated for selected jobs, adjusting content based on user's profile and job description, acting as an effective tool for efficient writing.
- Job Listings: Job board integration with job listings scraped from multiple sources.
- Subscription Model: Tiered subscription model with progressively advanced features.
- User Support: Self-help documentation and an AI assistant to answer basic user questions and to guide users on the use of the application.
- Referral Program: Referral program allowing users to earn application credits.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to evoke trust and professionalism, while remaining accessible.
- Background color: Light gray (#F0F4F8) for a clean and modern feel.
- Accent color: Teal (#4DB6AC) to highlight key actions and information.
- Headline font: 'Poppins' sans-serif for a contemporary, precise feel.
- Body font: 'PT Sans' sans-serif providing readability and warmth to text blocks.
- Simple, professional icons related to job searching and career management.
- Clean and intuitive layout with a focus on readability and ease of use.
